# 15년 특허 조사·분석 실행 가이드 (예시: 센서 융합 디스플레이 RFP)

**예시 RFP**: (2026)_RFP_센서_융합_디스플레이_기술.md  
**분석 기간**: 우선일 2011.01.01 ~ 2025.12.31 (지난 15년)  
**CSV**: 사용자가 Google Patents에서 검색 후 **직접 다운로드**하여 제공. (Playwright 자동 다운로드는 보류.)

---

## 1. 검색식 및 URL (예시)

**검색식**:
```
(Display OR Transformable OR Deformation OR sensing OR User OR interface OR User OR Experience OR display OR panel OR screen OR OLED OR LED OR flexible OR stretchable OR foldable OR rollable OR backplane OR TFT OR pixel)
```

**날짜 필터**: 우선일 2011.01.01 ~ 2025.12.31

**검색 URL** (`output/search_url_15yr.txt` 에도 동일):
```
https://patents.google.com/?q=%28Display+OR+Transformable+OR+Deformation+OR+sensing+OR+User+OR+interface+OR+User+OR+Experience+OR+display+OR+panel+OR+screen+OR+OLED+OR+LED+OR+flexible+OR+stretchable+OR+foldable+OR+rollable+OR+backplane+OR+TFT+OR+pixel%29&after=priority:20110101&before=priority:20251231
```

---

## 2. CSV 준비 (사용자)

1. 브라우저에서 위 **검색 URL**을 연다.
2. 결과 페이지에서 **Download (CSV)** 버튼을 클릭해 CSV를 저장한다.
3. 저장한 CSV를 프로젝트에 두고 경로를 확보한다 (예: `output/google_patents_results.csv`).

---

## 3. 파이프라인 실행 (CSV 제공 후)

**일반 파이프라인** (어떤 RFP·분야에도 사용 가능):

```bash
cd .codex\skills\patent-strategy-report\scripts
uv run python run_full_pipeline.py "..\output\google_patents_results.csv" --rfp-path "..\..\..\(2026)_RFP_센서_융합_디스플레이_기술.md" --report-title "센서 융합 디스플레이 세계 특허 현황 분석 (2011-2025)" --topic "센서융합디스플레이" --search-query "(Display OR Transformable OR ...)" --search-url "https://patents.google.com/?q=..."
```

또는 **이 예시 전용** 스크립트 사용 (동일 RFP·15년 고정):

```bash
uv run python run_15yr_pipeline.py "..\output\google_patents_results.csv"
```
